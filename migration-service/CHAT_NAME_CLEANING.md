# Очистка названий чатов

## Проблема

При импорте данных из Excel или MaxBot API в поле `name` чата могут попадать лишние кавычки и пробелы:

```json
{
  "chats": [
    {
      "id": 36539,
      "name": "'                                      ЮРм-С251ГПз, ЮРм-С251ГПзб'",
      "url": "https://max.ru/join/5QrKYAeCv81GNC8VUT1An7k8Nw0jt8wRFwtqDIY_cqc"
    }
  ]
}
```

## Решение

Добавлена функция `cleanChatName()` в `migration-service/internal/usecase/migrate_from_excel.go`, которая:

1. Убирает пробелы по краям строки
2. Убирает одинарные кавычки (`'`) по краям
3. Убирает двойные кавычки (`"`) по краям
4. Повторно убирает пробелы после удаления кавычек

## Примеры

| До очистки | После очистки |
|-----------|---------------|
| `'                                      ЮРм-С251ГПз, ЮРм-С251ГПзб'` | `ЮРм-С251ГПз, ЮРм-С251ГПзб` |
| `"Группа ИВТ-31"` | `Группа ИВТ-31` |
| `   Группа с пробелами   ` | `Группа с пробелами` |
| `  '  Название  '  ` | `Название` |

## Использование

Функция автоматически применяется при обработке данных из Excel:

```go
excelRow := ExcelRow{
    ChatName: cleanChatName(row[11]), // Колонка 11 - Название чата
    // ...
}
```
