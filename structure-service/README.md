# Structure Service

Сервис для управления иерархической структурой вузов.

## Описание

Сервис позволяет:
- Отображать структуру вуза в иерархическом виде: Вуз → Филиал → Факультет → Группа → Чат
- Импортировать структуру из Excel файла
- Управлять структурными подразделениями (создание, обновление, удаление)

## Структура данных

Иерархия:
- **University** (Вуз) - корневой элемент
  - **Branch** (Филиал) - опциональный уровень
    - **Faculty** (Факультет/Институт)
      - **Group** (Группа)
        - **Chat** (Чат) - связь с chat-service

## API Endpoints

### Университеты
- `GET /universities` - Получить список всех вузов
- `GET /universities/{id}` - Получить вуз по ID
- `POST /universities` - Создать новый вуз
- `GET /universities/{id}/structure` - Получить полную структуру вуза

### Импорт
- `POST /import/excel` - Импортировать структуру из Excel файла

## Формат Excel файла

Excel файл должен содержать следующие колонки:
- Номер телефона администратора
- ИНН
- ФОИВ
- Наименование организации
- Наименование головного подразделения/филиала
- КПП
- Факультет/институт/иная структурная классификация
- Курс обучения
- Номер группы
- Название чата
- Ссылка на чат

## Запуск

```bash
# Сборка и запуск через Docker Compose
make up

# Или вручную
docker compose up --build
```

Сервис будет доступен на порту `8081`.

## Swagger

Документация API доступна по адресу: `http://localhost:8081/swagger/`
