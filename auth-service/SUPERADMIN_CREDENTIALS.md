# Учетные данные супер админа

## Автоматически созданный супер админ

При инициализации auth-service автоматически создается пользователь с правами супер админа:

### Учетные данные
- **Телефон**: `+79999999999`
- **Email**: `superadmin@digitaluniversity.ru`
- **Пароль**: `tEst123!`
- **Роль**: `superadmin`

### Использование

```bash
# Авторизация через API
curl -X POST http://localhost:8080/login-phone \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+79999999999",
    "password": "tEst123!"
  }'
```

### Права доступа

Супер админ имеет полные права в системе:
- Управление всеми пользователями
- Создание и управление университетами
- Назначение ролей кураторов и операторов
- Доступ ко всем данным без ограничений по контексту

### Безопасность

⚠️ **Важно**: Рекомендуется изменить пароль после первого входа в продакшн среде.

### Техническая информация

Пользователь создается миграцией `000006_add_superadmin_user.up.sql` при инициализации базы данных.
